Account provisioning for Google Apps - Self account provisioning demo
=====================================================================

This demo provides a web front-end to:
1. Select a username from a list of auto-generated and available usernames
2. Create a Google Apps account in the configured domain

INSTRUCTIONS
---------------------------------------------------------------------
1. Run the appsProvisioning-0.0.1.jar RESTful API server. See README.md for
   instructions
2. Open self-provisioning-demo.js and set API_HOST to the address of the server
   hosting the RESTful API. e.g. http://localhost:8080
3. Open index.html
