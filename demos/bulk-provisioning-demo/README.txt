Account provisioning for Google Apps - Bulk account provisioning demo
=====================================================================

This demo provides a web front-end to create multiple Google Apps accounts in
the configured domain.

INSTRUCTIONS
---------------------------------------------------------------------
1. Run the appsProvisioning-0.0.1.jar RESTful API server. See README.md for
   instructions
2. Open bulk-provisioning-demo.js and set API_HOST to the address of the server
   hosting the RESTful API. e.g. http://localhost:8080
3. Optional: set a default password in DEFAULT_PASSWORD. This password will be
   used to create all the accounts.
4. Open index.html
